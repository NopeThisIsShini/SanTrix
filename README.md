# SanTrix

**A powerful, MIT-licensed rich text editor for the modern web.**

SanTrix is a professional-grade WYSIWYG editor designed for reliability,
power, and deep customization. It is built on a mature open-source editor
codebase and provides a fully MIT-licensed solution suitable for both
open-source and commercial use.
