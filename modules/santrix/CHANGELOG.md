# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## 0.0.1 - 2026-01-14

### Added

- Initial release of SanTrix, a rebranding of the HugeRTE/TinyMCE rich text editor.
- All core editor functionality including:
  - Rich text formatting (bold, italic, underline, strikethrough)
  - Lists (ordered, unordered)
  - Tables
  - Links and images
  - Code samples with syntax highlighting
  - Emoticons
  - Full-screen editing
  - Search and replace
  - Word count
  - And many more plugins

### Changed

- Rebranded from HugeRTE/TinyMCE to SanTrix
- Updated global object from `tinymce`/`hugerte` to `santrix`
- Fresh versioning starting from 0.0.1
